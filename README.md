# Projet-ANOVA
# 📊 Analyse de la Croissance des Plantes (Projet ANOVA)

## 📝 Description du Projet
Ce projet a été réalisé dans le cadre du cursus **ISE 2 (Ingénieur Statisticien Économiste)** à l'**ENSAE Dakar / ANSD**. L'objectif est d'étudier l'impact de différents traitements (engrais, conditions environnementales) sur la croissance végétale en utilisant des méthodes d'analyse de la variance.

Le rapport met en œuvre une démarche statistique rigoureuse, allant de l'exploration des données à la validation des hypothèses de Gauss-Markov.

## 🚀 Fonctionnalités
- **Analyse Descriptive** : Visualisation des distributions (Boxplots, Histogrammes).
- **Modélisation ANOVA** : Test de l'effet traitement sur la variable d'intérêt.
- **Validation du Modèle** : Tests de normalité (Shapiro-Wilk) et d'homoscédasticité (Bartlett/Levene).
- **Tests Post-hoc** : Comparaisons multiples de Tukey pour identifier les groupes significativement différents.
- **Design Professionnel** : Rapport généré en PDF via RMarkdown et LaTeX.

## 🛠️ Technologies Utilisées
- **Langage** : R
- **Outils** : RMarkdown, LaTeX (TinyTeX)
- **Librairies clés** : `ggplot2`, `dplyr`, `knitr`, `kableExtra`

## 📁 Structure du Dépôt
